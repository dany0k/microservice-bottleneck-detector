# microservice-bottleneck-detector

Программа динамически строит граф микросервисов, запущенных через Docker, и их связи.

## Возможности
- Сканирует запущенные контейнеры
- Определяет потенциальные связи между сервисами (HTTP probe)
- Строит интерактивный граф в виде HTML
